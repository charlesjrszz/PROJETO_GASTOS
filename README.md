# Fin Organizer (Electron)

Aplicativo desktop para organizar finanças pessoais.
Suporta importação de arquivos OFX, XLSX e ODS e exportação CSV. Usa `electron-store` para persistência local e pode ser estendido para backup no Firebase.

Como rodar:

1. npm install
2. npm start
